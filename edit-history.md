# 编辑记录

## 2026-05-15 准星安全区双模式优化

### 更改内容
- 准星安全区从单一遮罩模式扩展为两种模式：
  1. **区域遮罩**（原有）：在屏幕中心绘制黑色遮罩覆盖 ESP 内容
  2. **范围内不绘制**（新增）：对安全区范围内的实体，可选择跳过特定 ESP 元素的绘制（方框、骨骼、血条、护甲条、武器、名称、连线、视线、头部圆点、距离，可多选）

### 更改文件
- `MenuConfig.h` — 新增 SafeZoneMode 和 10 个 SafeZoneSkip* 配置变量
- `Language.h` — 新增模式选择和跳过选项的中英文字符串
- `GUI.cpp` — 安全区 UI 增加模式下拉框和跳过选项复选框
- `Cheats.cpp` — 将 2D 框计算提前，新增安全区内实体检测逻辑，每个 ESP 绘制调用添加跳过条件；遮罩模式仅在 Mode=0 时绘制
- `ConfigSaver.cpp` — 新增配置项的保存和加载

### 更改原因
用户需求：准星安全区支持两种模式，遮罩模式保留，新增可选择不绘制内容的跳过模式
